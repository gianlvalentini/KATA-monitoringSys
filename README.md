# KATA-monitoringSys

Desarrollo de la kata que emula un sistema de monitoreo de mediciones. 

Mas info del enunciado en la WIKI del proyecto.

# SOFTWARE REQUIRED

    JDK 8 (JDK8)
    Maven

# Installation

Para instalar las dependencias: 

    # cd proyecto
    mvn install
